Необходимо написать опросник с помощью React.


Правильный вариант ответа может быть один (radio) или несколько (checkbox).


Вопросы обладают аттрибутом "сложность" со значениями - easy, middle, hard.


Каждый следующий вопрос, отображается после ответа на предыдущий.


На финальном шаге отобразить результаты опроса, используя градацию по уровням сложности.


В качестве API можно использовать любой открытый источник, например: https://opentdb.com/api_config.php.


Для валидации кода предпочтительно использовать настроенный eslint https://www.npmjs.com/package/eslint-config-rambler
